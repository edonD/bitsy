================================================================================

1. Install dependencies:
   cd backend && poetry install

2. Start server:
   python main.py

3. Open in browser:
   http://localhost:8000/docs

4. Try a simulation:
   - Click POST /api/simulations/simulate
   - Click "Try it out"
   - Paste example JSON (see QUICK_START.md)
   - See instant prediction + SHAP breakdown!

Time to running: ~2 minutes

================================================================================
                         CORE FEATURES BUILT
================================================================================

🎯 What-If Simulation
   → User modifies features (add schema, publish article, etc.)
   → Model instantly predicts new mention rate
   → Returns lift in percentage points

📊 SHAP Explanations
   → Shows exact contribution of each modified feature
   → Helps user understand why prediction changed
   → Both global + local explanations

📈 Confidence Intervals
   → 95% confidence bounds on prediction
   → Accounts for model uncertainty
   → Shows "High" vs "Medium" confidence

🎪 Edge Cases
   → "What if only 1 authority source covers it?"
   → "What if content ages before getting traction?"
   → "What if competitors copy our strategy?"

🔍 Sensitivity Analysis
   → Vary each feature across a range (0.5x to 1.5x)
   → See how robust the prediction is
   → Identify key success factors
