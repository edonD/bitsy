Tab nav explorations — pick one

Ten directions for the five-section nav. Each row shows the same nav with Observe active.

01 · SEGMENTED PILL
Soft segmented control
Modern app-default. Sits in a sunken paper-2 well, active state floats as a paper card. Quiet, conventional, easy to read.
02 · UNDERLINE
Inline underline tabs
Classic horizontal tabs sitting on a hairline. Active = bold ink + 2px ink underline. Calm and document-feeling.
03 · INK SLAB
High-contrast active
No container, no underline. Just text — the active item is a solid ink rectangle. Confident, almost aggressive in its simplicity.
04 · NUMBERED RAIL
Engineering rail with step numbers
Bordered cells, mono step numbers, active item highlighted. Closest to the "loop / process" idea without being a wizard.
05 · STATUS DOTS
Dot-and-pill with live indicator
Each tab gets a small dot. Active dot pulses in rust (live data accent). Implies state without screaming progression.
06 · MASTHEAD CAPS
All-caps section markers
Tracked-out caps like a magazine masthead. Active gets a small rust top-mark. Editorial, less app-y.
07 · BUTTON GROUP
Connected slab buttons
Connected button-group with bold ink active state. Reads as a hard control — no ambiguity about what's selected.
08 · FOLDER TABS
Manila file folders
Literal folder-tab metaphor. Active tab lifts and merges with the content surface. Skeuomorphic but distinctive.
09 · KEY CHIPS
Hotkey hints — power-user energy
Each tab pairs with a keyboard chip (1–5). Implies fast switching, signals this is a tool not a marketing site.
10 · TICKER
Terminal/instrument bar
Black bar with mono caps. Active gets the rust accent — same color language as live data. Most "instrument-like" of the lot.
